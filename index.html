Common Issues

  • Microphone not working: Enable microphone in iOS Settings.
  • Import failed: Use mp3, m4a, wav, or m4r files.
  • No sound on preview: Check mute switch and volume.
  • Ringtone install help: Export the .m4r and transfer via Finder.
