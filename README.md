# Ringa Ringa Support Site

This folder contains a free static support website for your app.

## Files
- `index.html` -> Support page
- `privacy.html` -> Privacy policy page
- `styles.css` -> Shared design styles

## Update before publishing
Replace `sup.ringaringa@gmail.com` with your real email in:
- `index.html`
- `privacy.html`

## Free publish option: GitHub Pages
1. Create a new GitHub repo (example: `ringa-ringa-support`).
2. Upload all files from this folder to repo root.
3. In GitHub repo: `Settings -> Pages`.
4. Source: `Deploy from branch`, Branch: `main`, Folder: `/ (root)`.
5. Save. Your site will be live at:
   `https://<your-username>.github.io/ringa-ringa-support/`

Use these URLs in App Store Connect:
- Support URL: `https://<your-username>.github.io/ringa-ringa-support/`
- Privacy Policy URL: `https://<your-username>.github.io/ringa-ringa-support/privacy.html`
